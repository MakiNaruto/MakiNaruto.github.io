如何建立一个github pages

一个简单的github pages示例： MakiNaruto
参考文章：
一、Github Repository 的创建
二、本地搭建Git
三、同步到github，拥有属于你自己的github pages

这里是示例，我创建的Repository名字为MakiNaruto
1.右键git bash 输入git clone https://makinaruto.github.io/MakiNaruto.git
这一步是将你的git download到本地 完成后会出现名为MakiNaruto的文件夹
2.同步下来的文件夹，将你的html文件，js，css文件提交即可，纯html不依赖样式文件也可以
输入以下命令提交你的页面:

cd MakiNaruto
git add .
git commit -m "Add simple code"
git push

大功告成，查看一下 https://makinaruto.github.io/MakiNaruto/
注意：目前发现一个问题，git push 的主页文件第一次生成后，只能更新不能删除该文件然后新添加，会导致主页丢失。
